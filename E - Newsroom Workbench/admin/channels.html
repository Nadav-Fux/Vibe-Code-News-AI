בפיתוח · Phase 2

החיבור ל-Telegram + WhatsApp ייכנס כפונקציות נפרדות תחת functions/api/publish-channel/:

  • Telegram outbound: Bot API sendMessage לכל ידיעה עם channels.includes("telegram")
  • Telegram inbound: webhook → יצירת news draft
  • WhatsApp outbound: Business Cloud API · message template
  • WhatsApp inbound: webhook → news draft (אופציונלי)

סודות הבוטים יישמרו ב-Cloudflare Pages env vars (TELEGRAM_BOT_TOKEN, TELEGRAM_CHANNEL_ID, WHATSAPP_TOKEN, WHATSAPP_PHONE_ID) — לא ב-localStorage או בקליינט.
