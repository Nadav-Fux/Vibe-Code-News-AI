🔧 רישום webhook (העתק-הדבק) 
curl -sS -X POST \
  "https://api.telegram.org/bot<BOT_TOKEN>/setWebhook" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://new.nvision.me/api/telegram-webhook",
    "secret_token": "<SECRET>",
    "allowed_updates": ["channel_post", "message", "edited_channel_post"]
  }'

החלף את <BOT_TOKEN> ב-TELEGRAM_BOT_TOKEN ואת <SECRET> בערך של TELEGRAM_WEBHOOK_SECRET ש-Cloudflare Pages מחזיק. אחרי הרצה Telegram יחזיר {"ok":true,"result":true}.

WhatsApp — מתוכנן

Phase WhatsApp-A

סטטוס: מתוכנן · עוד לא פעיל

התוכנית עבור Phase WhatsApp-A:

  • Outbound: WhatsApp Business Cloud API — שליחת ידיעות דרך message template מאושר (כי הודעות לא-template מותרות רק בתוך חלון של 24 שעות).
  • Inbound: webhook ל-/api/whatsapp-webhook יקלוט הודעות נכנסות מהעורך וייצור news draft (אנלוגי ל-telegram-source).
  • סודות: WHATSAPP_TOKEN, WHATSAPP_PHONE_ID, WHATSAPP_VERIFY_TOKEN ב-Cloudflare Pages env vars.
  • ניתוב: ידיעה תכלול את הערוץ "whatsapp" ב-channels[] כדי שתישלח אוטומטית.
