תכנון מפורט

⏳ Phase 2B AI assist בעורך מאמרים

הוספת "סדר עם AI" לblock editor — לוקח טקסט גולמי ומחזיר בלוקים מובנים.

  • → כפתור ✨ "סדר עם AI" בסרגל v5e-editor-bar-r ב-v5-editor.jsx
  • → mode=article: שולח לapi/ai-format, response מפוצל לפי סעיפים → heading2 + paragraphs
  • → בלוק AI image generation: prompt → תמונה (Pollinations בברירת מחדל)
  • → תלות: functions/api/ai-format.js קיים ועובד (Phase 2A)
🗺️ Phase 3 Research + Multi-version generation

3.1 /api/research — חיפוש חי ממקורות מרובים

  • → Brave Search API: חינמי עד 2K/חודש
  • → Grok (X/Twitter): חיפוש ב-X בלבד
  • → SERP: יידרש CloakBrowser/proxy

3.2 Multi-version — הרצה מקבילה מול 2–3 LLMs

  • → UI: כרטיסיות A / B / C עם תוצאות
  • → הצבעה → הגרסה הנבחרת מוזרקת לeditor
  • → Providers: Groq (נוכחי) · Claude · OpenAI

3.3 AI Image Generation

  • → ברירת מחדל: Pollinations.ai — חינמי, ללא API key
  • → אלטרנטיבה: Cloudflare Workers AI flux-1-schnell-fp8
🗺️ Telegram Telegram bridge — track נפרד

שלושה כיוונים: שידור, קבלה, ייבוא היסטוריה.

  • → Outbound: publish עם channels.includes('telegram') → Bot API sendMessage לערוץ TELEGRAM_CHAT_ID
  • → Inbound webhook: POST /api/telegram-webhook → draft חדשות ב-Sanity
  • → Historical retrieve: Node.js script חד-פעמי, מייבא היסטוריית ערוץ ל-Sanity עם status="imported"
  • → Bot tokens כבר קיימים ב-KV: TELEGRAM_BOT_TOKEN · TELEGRAM_CHAT_ID · TELEGRAM_CMD_BOT_TOKEN · Ralph bots ×2
